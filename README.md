# BioSecureGate – Frontend User Interface

Final Year Computing Project (PUSL3190)  
BSc (Hons) Software Engineering  
Plymouth University (Delivered via NSBM Green University)

## Project Overview
BioSecureGate is a web-based biometric identity verification system designed for
airport and roadside security environments. The system supports facial and
fingerprint-based verification to improve identification accuracy and reduce
manual processing delays.

This repository contains the frontend user interface implementation of the system.

## Technologies Used
- Next.js (React Framework)
- TypeScript
- HTML5 and CSS3
- Git and GitHub

## Current Development Status
- Frontend UI design and layout implemented
- Role-based user interface for Admin and Officers
- Biometric enrollment and verification screens prepared
